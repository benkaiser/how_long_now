## How Long Until X

This module tells you how long until 5pm next passes (or any other specified time of day).

Useful for embedding in a statusbar or such.

## Installation

```
npm install -g how-long-until
```

## Usage

(example at 2:30pm)

```
$ how-long-until
2:30:00
$ how-long-until 18
3:30:00
$ how-long-until 18:20
3:50:00
```
